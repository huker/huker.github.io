attr and prop

遇到的问题：在获取checkbox的状态的时候，.attr()没有正确获取到checked的值。切换时候监听change事件，也没有随着改变

原因：dom对象的属性分两种，attribute和property。

列举一些常用的：

attribute -> .attr() property -> .prop()
src selected
href checked
class multiple
id readOnly
type ….
